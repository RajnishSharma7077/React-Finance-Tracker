## 💾 Data & Backup

### Data Storage
- **Method:** Browser localStorage
- **Location:** DevTools → Application → Local Storage
- **Persistence:** Remains after browser close
- **Capacity:** ~5-10MB per domain

### Backup Your Data
```javascript
// In browser console:
JSON.stringify(localStorage)
// Copy and save to file
```

### Reset Data
```javascript
// In browser console:
localStorage.clear()
// Refresh page
```
